# Скетчи для тестирования компонентов платы ESP32-C6-LCD-1.47.  

## Для работы с платой в Arduino IDE:  

1. Установить расширение esp32:  
	a. Tools/Board:<Наименование>/Boards Manager…  
	b. Найти  esp32  
	c. INSTAL  
2. Настроить плату ESP32-C6-LCD-1.47:  
	a. Выбрать плату: Tools/Board:<Наименование>/esp32/ESP32C6 Dev Module  
	b. Выбрать порт: Tools/Port:<Номер порта>  
	c. Если необходимо выводить сообщения в Serial Monitor: Tools/USB CDC On Boot: “Enabled”  
	d. Tools/Flash Size: “4MB (32Mb)”  
	e. Tools/Partition Scheme: “No OTA (2MB APP/2MB FATFS)”  

## Скетчи:  

### ButtonBoot  
- Переназначение кнопки boot в кнопку действия.  
- Выводит в Serial Monitor «Кнопка нажата» / «Кнопка отпущена».  
- Для работы:  
	Залейте скетч на плату  
	Откройте Serial Monitor  
	Нажмите и опустите кнопку boot  
### ButtonBootDiod  
- Переназначение кнопки boot в кнопку действия, для включения встроенного светодиода.  
- Включает/выключает встроенный светодиод и выводит в Serial Monitor «Кнопка нажата» / «Кнопка отпущена».  
- Для работы:  
	Залейте скетч на плату  
	Откройте Serial Monitor  
	Нажмите и опустите кнопку boot  
### DiodRGB  
- Переключение цветов встроенного светодиода.  
- Переключает цвета встроенного светодиода  
- Для работы:  
	Залейте скетч на плату  
### WiFiAP  
- Создание wifi точки доступа (AP).  
- Включает/выключает встроенный светодиод и выводит в Serial Monitor «Кнопка нажата» / «Кнопка отпущена».  
- Для работы:  
	Залейте скетч на плату  
	На устройстве (ПК, телефон) найдите WiFi сеть ServerESP32, с паролем 01234567  
	В браузере перейдите по адресу 192.168.2.1
